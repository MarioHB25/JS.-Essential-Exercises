Mini text editor

  • Volver

This Java Script example contains some tools for creating and editing text. This exercise uses conditional flow control structures and loops.

How does it work?

First open the element inspector and go to the console...

    Enter the number that corresponds to one of the following options
  • 0 - Stop program execution
  • 1 - Duplicate text
  • 2 - Replace one string with another in the text
  • 3 - Add text
  • 4 - Enter two numbers with indexes to trim the text
  • 5 - Delete a word from the text
  • 6 - Search for a string within the text and return a message saying whether it was found or not
  • 7 - Put all text in capital letters
  • 8 - Make all text lowercase
  • 9 - Show how many letters the text has without spaces
  • 10 - Show how many words the text has
  • 11 - Convert text to "Hacker Speak"
